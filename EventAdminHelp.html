Admin Guide

How To Use The EMS Admin Portal

This portal is the control center for event administrators. Use it to create and manage events, monitor attendance activity, review event performance, and keep your account details in one place. The sections below explain what each page is for and what work an admin should complete there.

dashboard

Dashboard

The dashboard gives you the quick overview of your current event activity. It shows the total number of events, ticket progress, upcoming events, revenue-style summaries, and the calendar. This is also the only place where new events should be created using the floating + button.

1
Review the summary cards first to understand total events, current ticket movement, and how many upcoming events are still active.
2
Use the floating + button to create a new event, fill in its details, upload its banner, and publish it into the system.
3
Watch the upcoming events section to confirm that live events are still future-facing and that past events automatically drop out once their time is over.
calendar_month

Events

The Events page is your browsing and management area. It separates events into Active, Past, and Draft states so you can inspect what is live, what is completed, and what still needs attention.

1
Use the status tabs to move between active, past, and draft events.
2
Use search and filters to quickly find a specific event by title, category, or time range.
3
Open any event card to view its details, registration progress, and approval status.
bar_chart

Attendee Insights

Attendee Insights is used for understanding how an event is performing. It summarizes attendee totals, location patterns, category trends, and demographic-style breakdowns for the currently selected event.

1
Open attendee insights when you want a focused view of one event’s participation trends.
2
Use the search field to switch between events and inspect the most relevant one quickly.
3
Use these insights while planning follow-up promotion, venue planning, and future event improvements.
badge

Profile, Notifications, And Session

The top-right account area is where an admin can check personal details and unique EMS profile ID. Notifications help surface recent activity, while logout ends the current working session safely.

Profile
Click the account box to open your profile viewer and see your unique EMS ID, email, role, and department details.
Notifications
Use the bell icon to review recent activity. If there is no new activity, the panel will clearly show that there are no new notifications.
Logout
Use logout when you finish working so the next person cannot access your admin session from the same machine.
task_alt

Recommended Admin Workflow

A simple routine helps admins keep the portal accurate and up to date throughout the day.

1
Start in the dashboard and review the live counts and upcoming events.
2
Create new events from the dashboard when needed, making sure the title, timing, venue, banner, and event category are all correct.
3
Move to the Events page to verify the event appears in the correct section and is easy to find through search or filters.
4
Use Attendee Insights to monitor how events are performing and to understand participation trends for planning and reporting.
5
Check notifications and your profile whenever needed, then log out once your work is complete.

If you face any issue while creating events, reviewing attendee information, or using your admin account, reach out to support and include the page name plus a short description of what went wrong.
